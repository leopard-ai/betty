Installation¶

Betty can be installed via pip. Betty’s dependencies include:

  • Python 3.6 - 3.9

  • PyTorch 1.6 - 1.12

Install with pip¶

pip install betty-ml

To use IterativeProblem, you need to additionally install functorch. However, functorch requires users to use PyTorch 1.11.

pip install functorch

To install Betty and develop locally:

git clone https://github.com/leopard-ai/betty.git
cd betty
pip install -e .

Verifying Installation¶

You can verify the installation by running the following:

python -c "from betty import test_install"

Additionally, you can verify the developer installation by running test.py.

python test.py

If the installation was successful, you should see the following output:
